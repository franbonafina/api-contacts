## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript for Contact API.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e

# e2e tests coverage
$ npm run test:e2e:cov
```
